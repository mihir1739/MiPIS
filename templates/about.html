MiPIS(Missing Person Identification System) is a Web-App/Website which can be used to help find the missing person around the globe. At the heart of it we have a Django server hosted on python anywhere having python in Backend and MySQL as default database. The front end is managed by Web3(HTML/CSS/JS) and the face recognition api is used from Face++ .The Algorithm used for finding similiarity is the popular algorithm known as K-Means Clustering Algorithm.
